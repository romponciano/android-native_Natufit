# Android Natufit

## en-us
Project developed for Natufit company. The purpose of the application is to allow the users to make their requests using the app. These requests are automatically sent to the company email that can assemble and take the order.<br>
https://play.google.com/store/apps/details?id=ponciano.romulo.natufit<br>
**Language:** Java/XML<br>
**IDE:** Android Studio<br>

## pt-br
Projeto desenvolvido para a empresa Natufit. O objetivo do aplicativo é permitir que os usuários possam realizar seus pedidos pelo app. Esses pedidos são enviados, automaticamente, para o email da empresa que poderá montar e levar o pedido.<br>
https://play.google.com/store/apps/details?id=ponciano.romulo.natufit<br>
**Linguagem:** Java/XML<br>
**IDE:** Android Studio
